All Features
Everything Pocket Shop offers out of the box
🔍

Searchable & Discoverable

Your shop lives on PocketWeb. Anyone can find it by keyword search or access it directly by address. No need to share links.

🔒

Encrypted Orders

Customer email, shipping address, and messages are encrypted with your Maxima public key. Only you can read them.

💳

Minima or USDT

Choose your currency at build time. Accept Minima or bridged USDT straight to your wallet.

🌐

Broadcast to Everyone

Published to PocketWeb, your shop is sent to every node on the network. No servers, no hosting, no gatekeepers.

🖥️

Studio Desktop App

Visual builder for macOS and Windows. No coding or terminal needed. Point, click, deploy.

⚡

Deploy in Minutes

From download to live shop in under 5 minutes. Fill in your details, click Build, publish to PocketWeb.

📬

mInbox Order Manager

Built-in vendor inbox MiniDapp. View orders, update status (Paid/Preparing/Shipped), export to CSV.

💬

ChainMail Integration

Reply to customers directly via encrypted ChainMail. Buyer's public key is included in each order.

🖼️

Smart Image Handling

Drag and drop product images. Studio auto-resizes to keep packages small. SVG, PNG, JPG supported.

💰

Zero Platform Fees

No monthly fees, no transaction cuts. Just standard Minima network fees.

💻

CLI Build Option

Prefer the terminal? Build shops from the command line with npx pocketshop or node build-pocketshop.js.

🪶

Ultra Lightweight

Shop packages stay under 50 KB. Tiny footprint, instant deploy. No bloat, no frameworks.

📦

npm Package

Install globally with npm i -g pocketshop. Run pocketshop to launch Studio anywhere.
